# UCCS 插件邏輯完整說明

> Unified Chinese-characters Searcher (UCCS) v2.0.0

---

## 🎯 核心目標

在 Obsidian 全局搜尋（`Cmd+Shift+F`）中，自動將輸入的中文字元擴展為多種地區寫法，確保輸入任何一個中文字都能找到所有對應中文的結果。

---

## 🧩 名詞解釋

| 名詞 | 說明 | 範例 |
|------|------|------|
| **s2t** | Simplified to Traditional — 簡體→繁體映射 | `里`→`裏`(HK) / `裡`(TW) |
| **t2s** | Traditional to Simplified — 繁體→簡體映射 | `裏`→`里` / `啟`→`启` |
| **格子** | 展開結果中的每個 `(文字)` 區塊 | `(里) OR (裏)` 有 2 格 |
| **防抖** | 等待用戶停頓後才展開，避免打字中途誤展開 | 預設 800ms |

---

## ⚙️ 完整流程

### 安全檢查（依序）

```
用戶打字
  → 插件正在更新？       是 → 跳過
  → 值和上次一樣？       是 → 跳過
  → 退格刪除（變短了）？   是 → 跳過
  → 游標不在末尾？       是 → 跳過
  → 有括號 ( ) ？        是 → 已展開，跳過
  → 有 OR 關鍵字？       是 → 複雜查詢，跳過
  → 沒有中文？           是 → 跳過
  → 字元無變體？
     ├ 短語關閉 → 跳過
     └ 短語開啟 → 有短語匹配？無→跳過 有→繼續
  → 全部通過 → 啟動防抖（預設 800ms）
                    ↓ 停頓後
  執行 _expandQuery(原文) → 展開
```

### 展開邏輯（_expandQuery）

```
原文 → tokenize 分詞
         ↓
  逐個 token 處理：
    type=plain + 有中文 → 字元轉換 → 短語補充
    type=operator + 開啟運算符轉換 → 轉換值
    其他 → 保留原文
         ↓
  組裝展開結果
```

## 🔤 字元轉換（核心）

### 方向規則

| 方向 | 是否受地區限制 | 說明 |
|------|--------------|------|
| **s2t**（簡→繁） | ✅ 是 | 決定用港版還是台版 |
| **t2s**（繁→簡） | ❌ 否 | 同時查 t2hk + t2tw + t2s，任何繁體都能找到簡體 |

### 單字元

```
getVariants("里")
  s2t: s2hk["里"]="裏"  s2tw["里"]="裡"  s2all["里"]=["裏","裡"]
  t2s: 同時查 t2hk+t2tw+t2s → "里"（但已過濾掉相同字）

  → HK: (里) OR (裏)
  → TW: (里) OR (裡)
  → ALL: (里) OR (裏) OR (裡)
  → HK↔TW: (里)  ← HK↔TW 只做繁體互轉，不處理
```

### 多字元

```
輸入「里里外外」
  逐字取 v[0] → "裏裏外外"（HK/預設版）
  ALL 模式再取 v[1] → "裡裡外外"（TW 版）

  → HK: (里里外外) OR (裏裏外外)
  → TW: (里里外外) OR (裡裡外外)
  → ALL: (里里外外) OR (裏裏外外) OR (裡裡外外)
```

## 📋 短語補充模組

```
字元轉換完成後
  短語開啟？ 查 stPhrases[文字]
    有匹配且不在 terms 中 → 追加
    無匹配或已在 terms 中 → 跳過

範例：
  老板 → 字元無結果 → 短語補「老闆」→ (老板) OR (老闆)
  一丝不挂 → 字元「一絲不掛」= 短語「一絲不掛」→ 跳過
```

## 🗺️ 四種映射模式 × 三種輸入 完整對照

| 輸入→ | 簡`里` | 繁HK`裏` | 繁TW`裡` |
|--------|---------|---------|---------|
| 簡↔HK 映射模式 | `(里) OR (裏)` | `(裏) OR (里)` | `(裡) OR (里)` |
| 簡↔TW 映射模式 | `(里) OR (裡)` | `(裏) OR (里)` | `(裡) OR (里)` |
| ALL 映射模式 | `(里) OR (裏) OR (裡)` | `(裏) OR (里) OR (裡)` | `(裡) OR (里) OR (裏)` |
| HK↔TW 映射模式 | `(里)` | `(裏)` | `(裡)` |

> **說明：**
> - 映射模式決定搜尋範圍，不會因輸入法不同而跑出範圍外的字
> - `getVariants` 只查一次映射，不會從簡體再回頭查第二次
> - `(字)` = 該字無變體，僅此一格

## 🔧 設定項

| 設定 | 說明 | 預設 |
|------|------|------|
| 啟用插件 | 一鍵開關 | 開啟 |
| 映射地區 | HK / TW / ALL / TW-HK | HK |
| 轉換運算符值 | 轉換 path:、tag: 等後的值 | 開啟 |
| 簡化搜索欄模式 | 搜索欄只顯示原文 | 關閉 |
| 展開延遲 | 打字停頓後多少 ms 展開 | 800ms |
| 短語/成語補充轉換 | 字符轉換不足時補充短語 | 關閉 |

### 4 種地區模式

| 模式 | 匹配方式 | 範例 |
|------|---------|------|
| 簡↔HK | `烟 / 菸` → `(烟) OR (菸)` |
| 簡↔TW | `启 / 啟` → `(启) OR (啟)` |
| ALL | `里 / 裏 / 裡` → `(里) OR (裏) OR (裡)` |
| HK↔TW | `裏 / 裡` → `(裏) OR (裡)` |
